## Chuks Kitchen – Food Ordering Web Application

Chuks Kitchen is a fully functional frontend food ordering web application that allows users to browse meals, manage a shopping cart, and complete a simulated checkout process.

The project was built to simulate a real-world e-commerce food delivery flow using Vanilla JavaScript and LocalStorage for state management.

### Key Features

User authentication (Sign Up & Sign In)

Dynamic meal browsing (Explore page)

Fully functional cart system:

Add to cart

Increase/decrease quantity

Remove items

Real-time total calculation

Delivery details form with validation

Payment page with card validation logic

Auto-generated unique Order ID

Order confirmation page

Persistent data storage using LocalStorage

Fully responsive design (mobile-first approach)

### Tech Stack
